## Reigo
## API to get informations about addresses

## Requirements :

selenium==3.141.0 

Flask==1.1.2 

numpy==1.18.5 

regex 

sqlite3

## Running
```shell
cd server
python API.py
```
Then go to http://127.0.0.1:5000/ and enter your address.

All the new entries will be saved in the database addresses.db
